---

## Wie du beitragen kannst

### Bug melden
→ [Issues → Bug Report](https://github.com/XED-dev/TUI/issues/new?template=bug_report.md)

Bitte angeben:
- Betriebssystem + Python-Version
- Was hast du erwartet? Was ist passiert?
- Wie kann man das reproduzieren?

### Feature vorschlagen
→ [Issues → Feature Request](https://github.com/XED-dev/TUI/issues/new?template=feature_request.md)

### Frage stellen
→ [Discussions → Q&A](https://github.com/XED-dev/TUI/discussions)

### Code beitragen

1. Issue öffnen und Idee beschreiben
2. Warten auf Feedback vom Maintainer
3. Fork + Branch anlegen: `feature/mein-feature`
4. Code schreiben (in deiner Muttersprache kommentieren — OK!)
5. Pull Request öffnen mit Referenz auf das Issue
