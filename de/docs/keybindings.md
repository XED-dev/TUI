# Tastenkürzel — XED /TUI
> Englische Version: `../../docs/keybindings.md`

## Navigation

| Taste | Aktion |
|---|---|
| `↑` / `k` | Eine Zeile hoch |
| `↓` / `j` | Eine Zeile runter |
| `K` / `J` | 5 Zeilen hoch / runter |
| `PgUp` / `PgDn` | Seite hoch / runter |
| `Home` / `g` | Erste Zeile |
| `End` / `G` | Letzte Zeile |
| `Tab` | Panel wechseln (→) |
| `←` / `→` | Panel wechseln |
| `Enter` | Session öffnen |

## Sessions

| Taste | Aktion |
|---|---|
| `a` | Claude Code `--resume` starten |
| `r` | `/resume <uuid>` in Clipboard |
| `s` | Sessions neu laden |
| `t` | Titel setzen |
| `d` | Session löschen (Bestätigung: "delete" tippen) |
| `/` | Live-Suche |

## Reader + Notizen

| Taste | Aktion |
|---|---|
| `e` | Notiz im `$EDITOR` öffnen |
| `c` | Notiz in Clipboard |
| `f` | Reader-Vollbild |
| `n` | Notiz-Vollbild |
| `m` | Panels tauschen (Move/Swap) |

## Layout

| Taste | Aktion |
|---|---|
| `0` | Top-Bereich ausblenden |
| `1`–`9` | Top-Höhe (n × 5 Zeilen) |
| `Ctrl+R` | Hot-Reload |

## System

| Taste | Aktion |
|---|---|
| `?` | Hilfe anzeigen |
| `q` / `Esc` | Zurück / Beenden |
