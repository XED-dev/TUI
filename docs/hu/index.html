XED /TUI

Böngésszen, keressen és folytassa Claude Code munkameneteit — közvetlenül a terminálban.

Telepítés

curl -fsSL https://tui.xed.dev/install.sh | bash

Szükséges: Claude Code · Python 3.11+ · Linux, macOS vagy WSL

Mit csinál

A Claude Code minden beszélgetést helyi fájlként tárol. Az XED /TUI hozzáférhetővé teszi őket: keressen az összes munkamenetben, olvasson teljes átiratokat Markdown megjelenítéssel, írjon jegyzeteket, és ugorjon vissza bármely munkamenetbe egyetlen billentyűvel.
