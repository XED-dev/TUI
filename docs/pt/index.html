XED /TUI

Navegue, pesquise e retome suas sessões do Claude Code — diretamente no terminal.

Instalar

curl -fsSL https://tui.xed.dev/install.sh | bash

Requer: Claude Code · Python 3.11+ · Linux, macOS ou WSL

O que faz

O Claude Code armazena cada conversa como um arquivo local. O XED /TUI os torna acessíveis: pesquise em todas as sessões, leia transcrições completas com renderização Markdown, escreva notas e volte a qualquer sessão com uma tecla.

Atalhos de teclado

TeclaAção
↑↓Navegar sessões
EnterLer transcrição completa
/Pesquisar em todas as sessões
aRetomar no Claude Code
eEscrever uma nota
?Ajuda (5 idiomas)

Por que XED /TUI?

RecursoXED /TUIAlternativas
Sem dependências✅ apenas Python stdlib❌ Go / Node.js
Notas por sessão✅ Sync inteligente❌ não disponível
Renderização Markdown✅ negrito, código, tabelas❌ texto simples
Ajuda multilíngue✅ DE/EN/FR/JA/ES❌ apenas inglês
Tempo de instalação✅ ~5 segundos❌ requer compilação
