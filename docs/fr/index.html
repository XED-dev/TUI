XED /TUI

Parcourez, recherchez et reprenez vos sessions Claude Code — directement dans le terminal.

Installation

curl -fsSL https://tui.xed.dev/install.sh | bash

Prérequis : Claude Code · Python 3.11+ · Linux, macOS ou WSL

Ce qu'il fait

Claude Code stocke chaque conversation sous forme de fichier local. XED /TUI les rend accessibles : recherchez dans toutes les sessions, lisez les transcriptions complètes avec rendu Markdown, prenez des notes et revenez à n'importe quelle session en une seule touche.

Raccourcis clavier

ToucheAction
↑↓Parcourir les sessions
EnterLire la transcription complète
/Rechercher dans toutes les sessions

eÉcrire une note
