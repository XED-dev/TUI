XED /TUI

Claude Code のセッションを閲覧・検索・再開 — ターミナルから直接。

インストール

curl -fsSL https://tui.xed.dev/install.sh | bash

必要環境: Claude Code · Python 3.11+ · Linux、macOS または WSL

機能概要

Claude Code はすべての会話をローカルファイルとして保存します。XED /TUI はそれらにアクセスしやすくします。全セッションを横断検索し、Markdown レンダリング付きで完全なトランスクリプトを読み、メモを書き、1 キーで任意のセッションに戻ることができます。

キーボードショートカット

キー操作
↑↓セッションを移動
Enter完全なトランスクリプトを読む
/全セッションを検索
aClaude Code で再開
eメモを書く
?ヘルプ（5 言語）

XED /TUI を選ぶ理由

機能XED /TUI代替ツール
依存関係ゼロ✅ Python 標準ライブラリのみ❌ Go / Node.js
セッションメモ✅ スマート同期❌ 非対応
Markdown レンダリング✅ 太字・コード・表❌ プレーンテキスト
多言語ヘルプ✅ DE/EN/FR/JA/ES❌ 英語のみ
インストール時間✅ 約 5 秒❌ ビルドが必要
