# Keybindings — XED /TUI
> German source: `../de/docs/keybindings.md`

## Navigation

| Key | Action |
|---|---|
| `↑` / `k` | One line up |
| `↓` / `j` | One line down |
| `K` / `J` | 5 lines up / down |
| `PgUp` / `PgDn` | Page up / down |
| `Home` / `g` | First line |
| `End` / `G` | Last line |
| `Tab` | Switch panel (→) |
| `←` / `→` | Switch panel |
| `Enter` | Open session |

## Sessions

| Key | Action |
|---|---|
| `a` | Start Claude Code `--resume` |
| `r` | Copy `/resume <uuid>` to clipboard |
| `s` | Reload sessions |
| `t` | Set title |
| `d` | Delete session (confirm by typing "delete") |
| `/` | Live search |

## Reader + Notes

| Key | Action |
|---|---|
| `e` | Open note in `$EDITOR` |
| `c` | Copy note to clipboard |
| `f` | Reader fullscreen |
| `n` | Notes fullscreen |
| `m` | Swap panels (move) |

## Layout

| Key | Action |
|---|---|
| `0` | Hide top area |
| `1`–`9` | Top height (n × 5 lines) |
| `Ctrl+R` | Hot-reload |

## System

| Key | Action |
|---|---|
| `?` | Show help |
| `q` / `Esc` | Back / Quit |
