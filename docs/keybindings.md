# Keybindings — XED /TUI
> German source: `../de/docs/keybindings.md`

## Navigation

| Key | Action |
|---|---|
| `↑` / `k` | One line up |
| `↓` / `j` | One line down |
| `K` / `J` | 5 lines up / down |
| `PgUp` / `PgDn` | Page up / down |
| `Home` / `g` | First line |
| `End` / `G` | Last line |
| `Tab` | Switch panel (→) |
| `←` / `→` | Switch panel |
| `Enter` | Open session |

## Sessions

| Key | Action |
|---|---|
| `a` | Start Claude Code `--resume` |
| `r` | Copy `/resume <uuid>` to clipboard |
| `s` | Reload sessions |
| `t` | Set title |
| `d` | Delete session (confirm by typing "delete") |
| `/` | Live search |
| `#` | Set tags (comma-separated) · filter with `/#tag` |

## Reader + Notes

| Key | Action |
|---|---|
| `e` | Open note in editor (auto-detect / configured via `Ctrl+E`) |
| `o` | Open note in default app (xdg-open / configured via `Ctrl+E`) |
| `u` | Update note with new turns (no editor) |
| `c` | Copy note to clipboard |
| `f` | Reader fullscreen |
| `n` | Notes fullscreen |
| `m` | Swap panels (move) |

## Layout

| Key | Action |
|---|---|
| `0` | Hide top area |
| `1`–`9` | Top height (n × 5 lines) |
| `Ctrl+R` | Hot-reload |
| `Ctrl+E` | Settings: editor and default app |

## System

| Key | Action |
|---|---|
| `?` | Show help (DE / EN / FR / JA / ES — switch with `←→`) |
| `q` / `Esc` | Back / Quit |
