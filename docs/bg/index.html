XED /TUI

Преглеждайте, търсете и продължавайте вашите сесии на Claude Code — директно в терминала.

Инсталация

curl -fsSL https://tui.xed.dev/install.sh | bash

Изисква: Claude Code · Python 3.11+ · Linux, macOS или WSL

Какво прави

Claude Code съхранява всеки разговор като локален файл. XED /TUI ги прави достъпни: търсете в сесиите, четете пълни транскрипти с Markdown рендериране, пишете бележки и се връщайте към всяка сесия с един клавиш.

Клавишни комбинации

КлавишДействие
↑↓Преглед на сесии
EnterЧетене на пълен транскрипт
/Търсене в сесиите
aПродължи в Claude Code
eНапиши бележка
?Помощ (5 езика)
